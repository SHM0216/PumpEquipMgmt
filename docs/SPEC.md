# 시스템 명세서 (SPEC)

## 1. 시스템 개요

**RPMS (Rainwater Pump Station Management System)** — 빗물펌프장 시설물·설비·**부품**·유지보수·점검을 통합 관리하는 웹 애플리케이션.

### 1.1 사용자
| 권한 | 역할 | 주요 기능 |
|------|------|----------|
| ADMIN | 시스템 관리자 | 사용자 관리, 마스터 데이터, 모든 기능 |
| MANAGER | 배수운영과 담당자 | 설비·부품·이력 CRUD, 보고서 |
| VIEWER | 부서장·유관부서 | 조회 전용 |
| FIELD | 현장 관리자 | 점검 입력 (Phase 8) |

### 1.2 운영 환경
- 브라우저: Chrome 120+, Edge 120+, Safari 17+ (IE 미지원)
- 해상도: 데스크톱 1280px+, 모바일 360px+
- 내부망 운영 (외부 인터넷 격리)

## 2. 핵심 개념

### 2.1 데이터 3계층 ★

```
Facility (펌프장/유수지/공통)
 ├── Equipment (호기 단위, optional)
 └── Part (부품·작업 단위, PRIMARY)
      └── Maintenance (계약 이력)
```

**Part는 유지관리의 실제 최소 단위이자 UI 기본 단위**. 예:
- 펌프장: 월성2
- 설비: 펌프
- 부품: `전동펌프 4호기`, `전동펌프 그랜드패킹·압력게이지`, `펌프 기초부`, `전동펌프 오일`

Equipment(호기 단위)는 선택적 — 일부 부품만 호기 지정.

### 2.2 시설물 구분 (Facility.kind)
- `pump` — 펌프장 (월성1/2/3)
- `reservoir` — 유수지 (대명유수지)
- `common` — 공통설비 (월성공통: 154KV 수전, UPS 등)

### 2.3 Part 상태 코드 (5단계) ★
- **정상주기** (11건) — 주기 내 정상 관리
- **누적지연** (17건) ★ — 주기 경과, 조속 대응 필요
- **신설설비** (9건) ◆ — 최근 도입
- **장기주기** (10건) — 10년+ 장주기, 상태기반
- **완료확인** (1건) ✓ — 최근 교체 완료 (계약이력 외)

### 2.4 Equipment 상태 (3단계)
- `good` (녹) / `warn` (주황) / `bad` (빨강)

### 2.5 예방정비 엔진
**Equipment 기반**: `lastMaintDate + PmMaster.cycleMonths` = 차기 예정일  
**Part 기반**: `Part.nextDueYear`, `Part.statusCode` 직접 사용

## 3. 페이지별 기능 명세

### 3.1 대시보드 (`/`)
- KPI 6카드: 관리 시설 / 관리 부품 / **★누적지연** / ◆신설 / 금년도 집행액 / 점검 건수
- 펌프장별 상태카드 (정상/주의/위험 + 경과년수)
- 대분류별 투입비 바 차트
- **부품 상태 도넛 차트** (정상/누적지연/신설/장기/완료)
- 예방정비 알림 Top 10
- 최근 유지보수 이력

### 3.2 시설물 현황 (`/facility`)
- 월성 1·2·3 종합 제원 비교표
- 시설별 카드 (설치년월, 전기용량, 펌프 제원)
- 공통 제원 (면적, 배수용량 등)
- 대명유수지·월성공통 별도 카드

### 3.3 설비대장 (`/equipment`)
- 필터: 시설/대분류/상태/검색
- 호기 단위 설비 목록
- 설비 카드 모달 3탭 (기본/이력/예방정비)
- CRUD

### 3.4 부품 관리 (`/parts`) ★ 신규 핵심
- 필터: 시설/설비타입/상태코드/검색
- 그룹핑 뷰: 펌프장 → 설비타입 → 부품 트리
- 각 부품 행: 이름, 규격, 주기, 차기시기, 상태
- **부품 카드 모달**:
  - 기본정보 (규격·주기·차기)
  - 이력 (historyText 파싱 + 연결된 Maintenance 목록)
  - 타임라인 (2020~2030 매트릭스)
- 필터 프리셋: "★누적지연만", "◆신설만", "당해년도 도래만"
- CRUD

### 3.5 부품 타임라인 (`/parts/timeline`) ★ 신규
- 2020~2030 11개년 × 48 부품 매트릭스
- 셀 아이콘: ● 시행, ◐ 예상, ★ 지연, ✓ 완료, ◆ 신설
- 펌프장 필터
- 색상으로 상태 표현

### 3.6 유지보수 이력 (`/maintenance`)
- 필터: 시설/대분류/연도/계약종류/검색
- 신규 등록 시 **partId 선택 UI** (추천: 같은 시설·분류의 Part 목록)
- 집계: 건수/합계/평균

### 3.7 예방정비 (`/pm`)
- 임박 PM 스케줄 (Equipment 기준)
- Part 기준 누적지연/당해 도래 목록
- 주기 마스터 관리

### 3.8 점검일지 (`/inspection`)
- 유형별 입력 (일일/주간/월간/정밀/수시)
- 대상·결과·내용·점검자

### 3.9 알림 (`/alerts`)
- Equipment 기반 알림 (3단계)
- **Part 기반 알림**: 누적지연 17건 + 당해 도래
- 그룹: 긴급/주의/예정

### 3.10 통계 보고서 (`/reports`)
- 연도별 집행 추이
- 시설별 투입비 구성
- 세부구분 TOP 10
- **부품 상태 요약** (펌프장별 × 상태)
- CSV 내보내기: 설비/**부품**/유지보수/점검

### 3.11 시스템 설정 (`/settings`)
- 분류 체계 / PM 주기 마스터 관리 / 데이터 관리
